map和parseInt

2020-05-30- 2022-06-02

前段时间看到了一个很经典的面试题。

它的结果为什么是[1,NaN,NaN]呢

接下来就展开分析一下

map函数

map函数是数组的一个方法，他会返回一个新数组，数组中的元素是原始数组调用函数处理后的值。

它会接受三个参数 Array.map((item,index,arr)=>{})

paeseInt

接受两个参数

string 必选。需要被解析的字符串

redix 可选。表示要把字符串以几进制解析（2-36）

paeseInt结合map函数，map函数会传递三个参数，而parseInt只能接受两个参数，因此map的第三个参数会被忽略。

也就变为了

parseInt(item,index)

item 是数组每一项来作为要被解析的字符串

index 是每一项的所有被作为解析的进制
