javascript数据类型

2019-08-13

javascript数据类型分类两大类：基本数据类型和引用数据类型

基本数据类型

数据类型 说明
null 空值，表示非对象
undefined 声明了变量但未赋值
number 数字，运算的值
string 字符串类型
boolean 布尔值，逻辑运算的值
symbol ES6新增的类型，表示唯一值

引用数据类型

数据类型 说明
Object 对象，由键值组成的无序集合
Array 数组，数据的集合
Function 函数，完成一段特定功能的代码块

存储方式

基本数据类型存在栈中，按照键值的方式存储，根据键取出对应的值。

引用数据类型存储在堆中，同时会在栈中保存堆的引用地址。
