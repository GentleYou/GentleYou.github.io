Object.create

2019-06-01- 2022-06-02

它是干嘛的

Object.create()方法用于创建对象，这个方法有两个参数。

prop：必选，表示新对象的原型对象。该参数可以是null，对象，函数的prototype属性。

propertiesObject：可选，新创建对象上的可枚举属性，包括对象的属性描述符和响应属性名称。

描述符
  • value：对象的默认值，默认为undefined。
  • configurable：设置是否可设置属性的特性，默认为false。
  • writable：设置属性是否可读，默认为false。
  • enumerable：设置属性是否可枚举，默认为false。
